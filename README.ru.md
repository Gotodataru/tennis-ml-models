Прогнозирование исходов теннисных матчей (ATP и WTA)

Этот репозиторий содержит полный набор моделей машинного обучения для прогнозирования исходов теннисных матчей по четырём рынкам: Победитель матча, Тотал геймов, Фора по геймам и Победитель первого сета. Модели обучены отдельно для ATP (мужчины) и WTA (женщины) с использованием продвинутого конструирования признаков и алгоритма CatBoost.



Проект демонстрирует:



Инжиниринг признаков на основе исторических данных матчей.



Обучение моделей с временной валидацией и изотонической калибровкой.



Подробные отчёты о качестве моделей и диагностические графики.



Чистый, модульный код для воспроизводимости.



⚠️ Только для образовательных целей: Проект предназначен для обучения и демонстрации. Модели показывают хорошие метрики на валидации, но бэктестинг на 10 000 матчах выявил отрицательную доходность для модели победителя. Реальные ставки связаны с финансовым риском; используйте на свой страх и риск.

📊 Подготовка данных

Скрипты обучения ожидают CSV-файлы со следующими колонками (точные названия могут отличаться; см. комментарии в каждом скрипте):



date – дата матча (используется для временного разделения)



gender – 'ATP' или 'WTA'



player1\_id, player2\_id – уникальные идентификаторы игроков



surface – покрытие корта (например, 'Clay', 'Hard', 'Grass')



winner\_1 – бинарная цель: 1 если победил первый игрок, иначе 0



total\_games – общее количество сыгранных геймов



games\_diff – геймы, выигранные первым игроком, минус геймы второго



first\_set\_bin – бинарная цель: 1 если первый игрок выиграл первый сет, иначе 0



плюс множество колонок-признаков (форма игроков, личные встречи, статистика по покрытиям и т.д.)



Поместите подготовленные файлы данных в папку data/. Основные скрипты обучения ожидают файлы с именами:



clean\_multimarket\_features.csv



winner\_features.csv



Обратитесь к комментариям в каждом скрипте для точного списка требуемых колонок.



🚀 Как воспроизвести модели

1\. Отбор признаков

Запустите feature\_selection.py для определения наиболее важных признаков для каждой цели и пола. Скрипт обрабатывает пропуски, удаляет константные и сильно коррелированные признаки, вычисляет важность с помощью CatBoost.



bash

python scr2/feature\_selection.py

Результаты (сохраняются в data/selected\_features/):



selected\_features\_<target>\_<gender>.txt – список отобранных признаков



feature\_importance\_<target>\_<gender>.csv – значения важности признаков



2\. Обучение финальных моделей

Запустите retrain\_final\_models.py для обучения всех восьми моделей (4 рынка × 2 пола) с использованием топ-11 признаков из предыдущего шага. Скрипт автоматически сохраняет модели, калибраторы, отчёты и диагностические графики.



bash

python scr2/retrain\_final\_models.py

Альтернативно можно обучать отдельные модели с помощью скриптов в scr2/ATP/ и scr2/WTA/, например:



bash

python scr2/ATP/train\_winner\_atp\_clean.py

3\. Результаты модели

Каждая обученная модель сохраняется в отдельную папку, например ATP/WINNER/model/winner\_atp/, содержащую:



model.cbm – обученная модель CatBoost



isotonic\_calibrator.pkl – калибратор (для классификационных моделей)



evaluation\_report.json – метрики на валидационной выборке



feature\_importance.png – график топ-20 важных признаков



learning\_curve.png – кривая обучения (потери на тренировке/валидации)



roc\_curve.png / residuals.png – диагностические графики



📈 Производительность моделей

Все модели валидированы на самых свежих 20% матчей (временное разделение). Вероятности откалиброваны с помощью изотонической регрессии, где указано.



Модели ATP

ECE (cal)	0.0125	Изотоническая калибровка

Тотал геймов	MAE	2.006	R² 0.808, RMSE 3.750

ECE (cal)	0.0125	Изотоническая калибровка

Фора геймов	MAE	3.217	R² 0.223, RMSE 4.903

ECE (cal)	0.0117	Изотоническая калибровка

Первый сет	ROC-AUC	0.6988	LogLoss 0.6261, Brier 0.2189

ECE (raw)	0.0063	Калибровка не требуется

Модели WTA

Модель	Метрика	Значение	Примечания

Победитель	ROC-AUC	0.7130	LogLoss 0.6107, Brier 0.2130

ECE (cal)	0.0144	Изотоническая калибровка

Ключевые наблюдения:



Модели ATP Победитель и ATP Тотал геймов показывают высокий уровень по сравнению с публичными бенчмарками.



Калибровка отличная (ECE <2% для всех), что критически важно для оценки вероятностей.



Фора геймов – самая слабая группа (R² ~0.2), что отражает высокую дисперсию разницы геймов.
